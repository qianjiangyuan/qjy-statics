文档开发指南

全局安装gitbook

npm i -g gitbook

起服务，在localhost:4000端口可以查看

gitbook serve

一般编辑可以直接在VS code里写markdown，也可以用可视化的编辑器，比如typora直接编辑。

开完完之后，可以通过build直接生成html文件，默认在_book的目录下。

gitbook build
